NAME
       mv - move a selected file to a selected location.

SYNOPSIS
       mv [file_name] [new_location]

DESCRIPTION
       Move an existing file to a new location by typing the file and the location as arguments.

       closedir(dirp) - This function  closes  the directory stream associated with dirp.  A successful call to closedir()  also closes the underlying file descriptor associated with dirp.  The directory stream descriptor dirp is  not available after this call.
